README

说明

全局安装 gitbook 和 gitbook-cli
npm install gitbook -g
npm install gitbook-cli -g
clone两个项目到本地

  • docs 作为原始md文件，每次编写前，pull此项目，在此项目编辑，生成Notepad项目

  • Notepad gitbook生成的html，releases版

git clone https://github.com/huangnan1/docs.git
git clone -b -gh-pages https://github.com/huangnan1/Notepad.git
docs

  • 等于web开发中代码库

  • SUMMARY.md 为目录，每次增加新章节，创建新的md文件，请在目录中添加对应链接

gitbook serve ，启动本地服务，在浏览器中查看效果

gitbook build ，生成_book目录， 此目录内容copy至 Notepad项目中发布

Notepad

  • 等于web开发中打包后的目录

  • 由docs项目build之后，覆盖至本目录下，然后推送至远程仓库

git add .

git commit -m '本次提交的概述'

git push origin gh-pages
